# Remind Me - React App
Check this project on vercel - [Remind Me](remind-16o9610mf-xshiro2.vercel.app)
## Built using:
- reactjs
- tailwindcss

## Getting started
#### 1. Access directory
```
cd remind-me
```
#### 2. Install packages
```
npm install --save --force
```
#### 3. Start app
```
npm start
```
