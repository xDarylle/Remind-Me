# Remind Me - React App
Check this project on vercel - [Remind Me](https://remind-me-git-main-xshiro2.vercel.app/)
## Built using:
- reactjs
- tailwindcss

## Getting started
#### 1. Access directory
```
cd remind-me
```
#### 2. Install packages
```
npm install --save --force
```
#### 3. Start app
```
npm start
```
